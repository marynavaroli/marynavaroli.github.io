# Mary Navaroli Resume Website

React resume website for Mary Navaroli, including education, experience, hobbies, project demos, and contact information.

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run optimize-images
npm run build
```

## GitHub Pages

The deployment workflow is in [.github/workflows/deploy.yml](/marynavaroli.github.io/.github/workflows/deploy.yml).

In the repository settings on GitHub, Pages should be set to deploy from `GitHub Actions`.
